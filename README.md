# ryuo-display

Linux tools for the ASUS ROG Ryuo I 240 AIO OLED display.

- `ryuo-oled.py` — CLI tool to upload GIFs/images to the pump head OLED
- `ryuo-gallery.py` — Desktop gallery app for previewing and applying GIFs
- `ryuo-daemon.py` — REST API daemon
- `99-ryuo.rules` — udev rules for device access
- `ryuo-oled.service` — systemd service
- `liquidctl-patch/` — liquidctl driver patch
- `archive/` — old probe/test scripts
